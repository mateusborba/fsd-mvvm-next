# FSD NEXT JS
Esse projeto tem como finalidade o estudo e aplicação da arquitetura Feature Sliced Design em projetos front-end.

# Tech Stack
- Nextjs na versão mais atual
- TanStack query
- React-hook-form
- Shadcn-ui, para componentes visuais

# Regras
- Aplicar sempre alterações seguindo o padrão da arquitetura Feature Sliced Design
- link de referência: https://feature-sliced.design/docs/get-started/overview
- Não crie barrel files
- Componentize todos os componentes, e mantenha a lógica sempre em outro arquivo utilizado como hook no componente de ui


# Skills
- Utilize sempre clean code e solid seguindo as rules do projeto
